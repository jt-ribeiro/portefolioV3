# Portfólio V3 — Tomás Ribeiro

Portfólio pessoal de Tomás Ribeiro, Fullstack Developer.

Construído com Next.js 16, Framer Motion e CSS Modules — com um design Premium Dark Mode focado em glassmorphism e micro-animações.

## Stack

- **Framework:** Next.js 16 (App Router)
- **Animações:** Framer Motion
- **Estilos:** CSS Modules (Vanilla CSS)
- **Fontes:** Inter + Outfit (Google Fonts)

## Páginas

| Rota | Descrição |
|------|-----------|
| `/` | Hero com partículas animadas e typewriter |
| `/about` | Bento grid com bio e valores |
| `/projects` | Cards interativos com accordion |
| `/skills` | Barras de progresso animadas |
| `/contact` | Formulário + métodos de contacto |

## Desenvolvimento

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
